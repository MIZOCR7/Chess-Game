# My Pygame Chess Game

Hey! Welcome to my Chess Game. I built this in Python using the pygame library with smooth piece movement, valid chess rules, and an interactive board. This game was made as a practice project to improve my Python knowledge and enhance my coding experience. It supports two-player gameplay and includes all the basic chess mechanics.

## How To Run

Just download the `.exe` from the Releases section and run `main.exe`.

## Controls

* **Select Piece:** Left Mouse Click
* **Move Piece:** Left Mouse Click on a valid square

## Why I Built It

I built this game to practice object-oriented programming (OOP), game logic, and handling complex rules in Python. It helped me improve my problem-solving skills and gain more experience building games with pygame.

## What Used to Build It

I used the **pygame** library, a Python module for creating games, to build the user interface, handle player input, and render the chessboard and pieces. The game uses classes and OOP principles to manage the pieces, game rules, and overall gameplay.
